quick Start

getting started with debian-firstb00t is straightforward

➡ simply download and run the script at the very first boot of your server to implement comprehensive security measures

# download debian-firstb00t script
wget https://github.com/debian-firstboot/script.sh

# make it executable
chmod +x script.sh

# run with root (sudo user will be created later in the process
./script.sh
